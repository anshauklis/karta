# Export

## CSV Export

Available from any chart card or SQL Lab:

1. Click the **CSV** button on a chart card
2. The file downloads with column formatting applied
3. Filename format: `{chart_title}.csv`

## Excel Export

Available from table and pivot table chart types:

1. Click the **Excel** button on a chart card
2. The exported file includes:
   - Number formats matching your column formatting configuration
   - Conditional formatting colors (background and text)
   - Auto-width columns based on content
   - Bold header row
3. Filename format: `{chart_title}.xlsx`

## PDF Export

Export an entire dashboard as a PDF:

1. Open a dashboard in view mode
2. Click the **PDF** button in the header
3. The dashboard is captured as-is and saved as an A4 landscape PDF
4. Filename format: `{dashboard_title}.pdf`

:::{note}
PDF export captures the current state of the dashboard including all rendered charts. Large dashboards may take a few seconds to process.
:::
