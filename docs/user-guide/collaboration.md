# Collaboration

## Sharing Dashboards

Share a read-only view of your dashboard with anyone — no account required.

### Creating a Share Link

1. Open a dashboard
2. Click **Share** in the header
3. Configure the link:
   - **Expiration** — optional: set the link to expire after N hours
4. Click **Create Link**
5. Copy the generated URL and share it

### What Shared Users See

- Full dashboard layout with all charts rendered
- Interactive charts (hover tooltips, zoom)
- No edit capabilities, no sidebar navigation
- A "Shared View" badge indicates the read-only mode

### Managing Share Links

1. Open the Share dialog on a dashboard
2. View all active share links with creation date and expiration
3. Click **Revoke** to immediately disable a share link

## Comments & Annotations

### Chart Comments

1. On any chart card, click the **Comments** icon (speech bubble)
2. A panel slides in from the right
3. Type your comment and press {kbd}`Enter` or click Send
4. Comments are visible to all users with access to the dashboard
5. Delete comments by clicking the trash icon

### Dashboard Comments

1. In the dashboard header, click **Comments**
2. Works the same as chart comments but applies to the dashboard as a whole
3. Use dashboard comments for high-level notes, status updates, or discussion

## Stories

Stories let you create narrative presentations combining charts, text, and annotations.

1. Navigate to **Stories** in the sidebar
2. Click **New Story**
3. Add slides with:
   - Charts from any dashboard
   - Rich text descriptions
   - Annotations highlighting specific data points
4. Present stories in sequence for stakeholder meetings or reports

## Change History

Every chart and dashboard tracks its change history automatically.

1. Open a chart in the editor
2. Click **History** in the toolbar
3. View a timeline of all changes:
   - Who made the change
   - When it was made
   - What type of change (create, update, duplicate)

## Bookmarks & Favorites

### Favorites

- On the home page, hover over a dashboard card and click the **star** icon
- Favorited dashboards appear in the **Favorites** section of the sidebar
- Favorites are stored per-browser (localStorage)

### Bookmarks

- Bookmarks provide server-side saved links to dashboards and charts
- Access bookmarks from the sidebar
