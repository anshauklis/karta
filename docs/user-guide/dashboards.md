# Dashboards

Dashboards are the primary way to organize and view your charts. Each dashboard has a grid layout where charts can be positioned and sized freely.

## Creating a Dashboard

1. Go to the **Home** page
2. Click **New Dashboard**
3. Enter a title and click **Create**
4. You'll be redirected to the dashboard view

## Dashboard View

The dashboard view displays all charts in a read-only grid layout:

- **Auto-execute** — all charts run their queries automatically on page load
- **Filter bar** — apply global filters that affect all charts on the dashboard
- **Cross-chart filtering** — click a data point on one chart to filter other charts by that value
- **Drill-down** — click a bar or slice to drill into that category. Breadcrumbs show active drill filters with one-click removal.
- **Refresh** — re-execute individual charts or the entire dashboard
- **Fullscreen** — expand any chart to a fullscreen dialog for detailed viewing

## Editing a Dashboard

1. Click the **Edit** button in the dashboard header
2. In edit mode you can:
   - **Add charts** — click "New Chart" to create a new chart in this dashboard
   - **Rearrange** — drag charts by their title bar to reposition them
   - **Resize** — drag the bottom-right corner of any chart to resize
   - **Edit chart** — click the pencil icon on any chart to open the chart editor
   - **Duplicate** — click the copy icon to create a copy of any chart with all its configuration
   - **Delete** — remove charts from the dashboard
3. Layout changes are saved automatically

## Dashboard Settings

- **Title** — the dashboard name shown in the sidebar and header
- **Description** — rich text description shown via the info button
- **Icon** — emoji icon displayed next to the title

## Dashboard Actions

| Button | Action |
|--------|--------|
| **Comments** | Open the dashboard-level comments panel |
| **PDF** | Export the entire dashboard as a PDF (A4 landscape) |
| **Share** | Create a public read-only link |
| **Edit** | Switch to edit mode |
