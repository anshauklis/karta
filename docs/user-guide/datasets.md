# Datasets

Datasets are saved SQL queries that can be reused across multiple charts.

## Creating a Dataset

1. Navigate to **Datasets** in the sidebar
2. Click **New Dataset**
3. Select a connection and write your SQL query
4. Give the dataset a name and description
5. Click **Save**

## Using Datasets

When creating a chart, you can select a dataset instead of writing SQL from scratch. This ensures consistency across charts that use the same data.

## Benefits

- **Single source of truth** — update the query in one place, all charts reflect the change
- **Consistency** — multiple charts from the same dataset always show the same data
- **Reusability** — avoid duplicating complex SQL across charts
