# Filters & Drill-Down

## Dashboard Filters

Dashboard filters apply to all charts simultaneously:

1. Open a dashboard in view mode
2. The **filter bar** appears below the header (if filters are configured)
3. Select filter values — all charts re-execute with the applied filters
4. Clear filters to return to the unfiltered view

## Cross-Chart Drill-Down

Click data points on charts to filter other charts on the same dashboard:

1. Click a bar, slice, or data point on any chart
2. A **drill breadcrumb** appears below the filter bar showing the active drill filter (e.g., `region = Europe`)
3. All other charts on the dashboard re-execute with this filter applied
4. Click the **X** on a breadcrumb to remove that drill filter
5. Click **Clear all** to remove all drill filters

Drill-down uses the chart's X axis column as the filter key.
