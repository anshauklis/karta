# User Management

Accessible from **Admin > Users** in the sidebar (admin role required).

## Roles

| Role | Permissions |
|------|-------------|
| **Admin** | Full access: manage users, connections, RLS rules, all dashboards |
| **User** | View and edit dashboards they have access to, SQL Lab, create charts |

## Managing Users

1. Navigate to **Admin > Users**
2. **Invite** — send an invitation link for new users
3. **Edit** — change role or deactivate users
4. **Delete** — remove a user account

## First Admin

The first user to register after installation automatically receives the **admin** role. Subsequent users receive the **user** role by default.
