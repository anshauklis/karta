# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| {kbd}`Cmd+K` / {kbd}`Ctrl+K` | Open command palette |

## Chart Editor

| Shortcut | Action |
|----------|--------|
| {kbd}`Cmd+Enter` / {kbd}`Ctrl+Enter` | Preview (run query and render chart) |
| {kbd}`Cmd+S` / {kbd}`Ctrl+S` | Save |
| {kbd}`Cmd+Shift+S` / {kbd}`Ctrl+Shift+S` | Save & Close |
| {kbd}`Cmd+Z` / {kbd}`Ctrl+Z` | Undo (chart configuration) |
| {kbd}`Cmd+Shift+Z` / {kbd}`Ctrl+Shift+Z` | Redo (chart configuration) |
| {kbd}`Escape` | Back to dashboard edit view |

## Tables

| Action | Trigger |
|--------|---------|
| Sort ascending | Click column header |
| Sort descending | Click sorted column header again |

:::{note}
Keyboard shortcuts are disabled when focus is inside the Monaco SQL editor or an input field to avoid conflicts with editor shortcuts.
:::
